# Demo - Các Vấn Đề Hiện Đại trong Data Mining

Chương trình demo tổng hợp cho 10 chủ đề trong bài giảng "Các Vấn Đề Hiện Đại" - Data Mining & Applications.

## Cấu trúc

```

├── index.html          # Trang chủ chọn chủ đề

├── demo4.html         # Online Advertising
├── demo5.html         # Sentiment Analysis
├── demo6.html         # Recommendation Systems
```

## Cách sử dụng

1. Mở file `index.html` trong trình duyệt web
2. Chọn chủ đề muốn xem demo
3. Tương tác với các tính năng demo

## Các tính năng demo

### 1. Information Extraction
- Named Entity Recognition (NER)
- Relation Extraction
- Event Extraction
- Highlighting và visualization

### 2. Topic Modeling
- LDA (Latent Dirichlet Allocation)
- Topic visualization
- Document-topic distribution

### 3. Link Analysis
- PageRank Algorithm với visualization
- HITS Algorithm (Hubs & Authorities)
- Interactive web graph

### 4. Online Advertising
- CTR Prediction
- GSP Auction simulation
- Revenue calculation

### 5. Sentiment Analysis
- Naive Bayes
- SVM
- LSTM
- BERT
- Batch analysis
- Model comparison

### 6. Recommendation Systems
- User-based Collaborative Filtering
- Item-based Collaborative Filtering
- Matrix Factorization
- Rating prediction

### 7. Social Network Analysis
- Centrality Measures (Degree, Betweenness)
- Network visualization với D3.js
- Interactive graph

### 8. Multimedia Data Mining
- Image Classification
- Feature extraction
- Confidence scoring

### 9. Network Data Analysis
- Traffic Prediction (ARIMA, LSTM)
- Anomaly Detection (z-score)
- Time series visualization

### 10. Security Risk Analysis
- Risk Scoring (Threat × Vulnerability × Impact)
- Phishing Detection
- URL analysis

## Yêu cầu

- Trình duyệt web hiện đại (Chrome, Firefox, Edge, Safari)
- Kết nối Internet (để load các thư viện CDN):
  - Font Awesome
  - Chart.js
  - D3.js
